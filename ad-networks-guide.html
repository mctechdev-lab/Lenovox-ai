HOW TO SET UP ADSTERRA (Step by Step)
1
Create Publisher Account
Go to Adsterra and click "Sign up as Publisher". Use your email. Choose Publisher (not Advertiser).
→ publishers.adsterra.com/register
2
Add Your Website
Dashboard → Sites → Add Site → Enter your Vercel URL (e.g. lenovoxai.vercel.app). Category: Education. They will review and approve within 24–72 hours.
3
Create Ad Units
After approval: Sites → Your Site → Create Ad Code. Choose these formats for Lenovox:
• Display Banner (300×250) — put in dashboard
• Social Bar — looks like push notification, great for mobile
• In-Page Push — for rewarded-style notifications
4
Copy Your Publisher Key and Ad Zone IDs
Each ad unit gets a unique Zone ID. Copy these — you'll need them in your code. Also copy your API key from Account → API.
5
Add to Lenovox
Replace or add alongside your AdSense code. Adsterra banner code looks like this:

HOW TO SET UP PROPELLERADS
1
Register as Publisher
Sign up with email. Choose "Publisher". Instant approval — no waiting.
→ publishers.propellerads.com/signup
2
Add Your Site & Create Ad Zones
Sites → Add Site → add your Vercel URL. Then create these zones for Lenovox:
• Interstitial — shows between page transitions (great for gaming section)
• Push Notification — users opt-in, you get paid per subscriber click
• OnClick / Popunder — earns when user clicks anywhere on page
3
Get Your Zone ID and API Token
Dashboard → API → Generate Token. Each zone has a unique ID you use in the ad script. Save these as Vercel env vars.
4
Add the Push Notification Subscribe Script
This is the most valuable — users subscribe once and you earn every time they click a push notification, even when they're not on your site.
Keys You Will Get
Zone IDSites → Your Zone → Zone ID
API TokenAccount → API Tokens → Create

HOW TO SET UP MONETAG

4
Set Payment Method
